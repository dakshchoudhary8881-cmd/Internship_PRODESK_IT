# 🍔 Food Truck Menu Management System

A responsive React application that helps food truck businesses digitally manage and organize their menu items — built with a clean, enterprise-grade monochrome interface.

## ✨ Features

- Dashboard overview with live statistics
- Add, search, and filter menu items in real time
- Automatic category-based icons
- Empty state and loading skeleton handling
- Form validation with duplicate detection
- XSS input sanitization
- Accessible, fully responsive UI

## 🔍 How It Works

- Menu starts empty on first launch
- Users add items through a validated form; duplicates are rejected
- Category icons are assigned automatically
- Search and filters update instantly, and the dashboard reflects changes in real time
- All inputs are sanitized before entering state
