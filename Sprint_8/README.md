## ✨ Overview

CineVerse AI is a modern **Netflix-inspired Movie Discovery Platform** built with **React 19** and powered by the **TMDB API** and **Google Gemini AI**.

Instead of relying on traditional keyword searches, CineVerse understands the **user's mood**, transforms it into an intelligent recommendation, and presents a curated collection of movies in a premium streaming-style interface.

Designed with performance, scalability, and user experience in mind, the application features **infinite scrolling**, **debounced search**, **persistent favorites**, **lazy loading**, and an **AI-powered Mood Discovery Engine**.

---

# 🚀 Core Features

- 🎬 Browse Popular Movies in real time using TMDB
- 🔍 Lightning-fast Movie Search with Debouncing
- 🤖 AI Mood Discovery powered by Google Gemini
- ❤️ Persistent Favorites with Local Storage
- ♾️ Infinite Scroll using Intersection Observer
- 🖼️ Native Lazy Loading for Posters
- 📱 Fully Responsive Netflix-inspired UI
- ⚡ Optimized React Performance
- 🎞️ Detailed Movie Information
- 🌙 Modern Dark Theme Interface

---

# 🧠 AI Mood Discovery

Unlike a traditional movie search, CineVerse allows users to describe how they feel.

The application intelligently interprets the mood using **Google Gemini AI**, identifies the most suitable movie category, and discovers a curated collection of relevant films through the TMDB Discover API.
