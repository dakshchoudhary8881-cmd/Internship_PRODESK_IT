**Live Demo:** [https://internship-prodesk-it.vercel.app]

## Overview

A single-page site built for Sprint 01: a hero that gets straight to the point, a services grid, company stats, and a contact form — wrapped in a sticky glass navbar with a smooth light/dark switch. No frameworks, no build step — just HTML, CSS, and JS doing the work.

## Features

- **Frosted-glass navbar** — sticky on scroll, with a real `backdrop-filter` blur
- **One-click theme switch** — light/dark mode driven by CSS variables, no page reload
- **Mobile-first navigation** — links collapse into a hamburger menu on small screens
- **Services grid** — three cards that go from 3 columns to 1 as the screen narrows
- **Hover micro-interactions** — buttons scale up, service cards lift on hover
- **Contact form UI** — name, email, and message fields (front-end only for now — not wired to a backend yet)

## Tech Stack

HTML5 · CSS3 (Flexbox, Grid, custom properties, `backdrop-filter`) · Vanilla JavaScript
